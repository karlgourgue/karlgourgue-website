# Repo Guide

## Structure
- index.html: live site source
- mockup-*.html: design variants, not deployed
- prototype-index.html: staging for ideas before merging into index.html
- resume.md: resume source text
- docs/: specs, learnings, patterns, roadmap

## Scripts
- Local preview: `python3 -m http.server 8000`

## Code Style
- HTML: semantic, 2-space indent
- CSS: inline in `index.html`, CSS variables at top
- JS: minimal, vanilla
- Keep files under ~500 LOC
