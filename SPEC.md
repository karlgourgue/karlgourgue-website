# Karl Gourgue - Personal Website Spec

## Overview
A clean, minimal personal website showcasing work background and side projects.

## Tech Stack
- **Plain HTML/CSS/JS** - No build step, simple deployment, easy to maintain

## Target Audience
- Potential employers/collaborators
- People interested in side projects
- Professional network connections

## Content Sections

### 1. Hero/Header
- Name: Karl Gourgue
- Brief tagline (role: Sales professional in LMS/EdTech space)
- Email contact link

### 2. About
- Professional background in sales for LMS solutions
- Personal interests/what drives you
- Keep it concise (2-3 paragraphs max)

### 3. Projects (1-3 initially)
Each project card should include:
- Project name
- Brief description (1-2 sentences)
- Tech used / what you built
- Link to live project or GitHub (if applicable)
- Screenshot or icon (optional)

### 4. Footer
- Simple email link
- Copyright/year

## Visual Design

### Style: Clean & Minimal
- Generous whitespace
- Simple, readable typography (system fonts or Google Fonts like Inter, Source Sans)
- Limited color palette (2-3 colors max)
- No clutter - let content breathe

### Responsive
- Mobile-first approach
- Works well on all screen sizes
- Simple single-column layout on mobile

## Technical Requirements
- Single `index.html` file (or simple multi-page if needed)
- Separate `styles.css` for styling
- Minimal or no JavaScript (only if needed for interactivity)
- Fast loading (<1s)
- Accessible (semantic HTML, good contrast)
- SEO basics (meta tags, title, description)

## Deployment Options
- GitHub Pages (free, easy)
- Netlify (free tier)
- Vercel (free tier)

## Content To Gather
- [ ] Professional bio/about text
- [ ] 1-3 project descriptions
- [ ] Project links/screenshots
- [ ] Email address for contact
- [ ] Headshot photo (optional)

## Out of Scope (for v1)
- Blog
- Contact form
- Analytics
- CMS/admin panel
